# 九宫格潜力展示系统 - 安装使用说明

## 📦 应用简介
这是一个基于Python开发的九宫格潜力数据展示工具，可以读取Excel文件并生成直观的3x3网格布局展示。

## 🚀 快速开始

### 方法一：直接运行打包应用（推荐）
1. **获取应用文件夹**
   - 将整个 `九宫格潜力展示系统` 文件夹复制到您的电脑上
   - 文件夹包含：
     - `九宫格潜力展示系统`（可执行文件）
     - `_internal/`（依赖文件夹）
     - 其他必要文件

2. **运行应用**
   - 双击 `九宫格潜力展示系统` 可执行文件
   - 或在终端中运行：`./九宫格潜力展示系统`
   - 应用会自动启动并在浏览器中打开

3. **使用应用**
   - 在网页界面中上传您的Excel文件
   - 系统会自动生成九宫格展示

### 方法二：源码运行（开发者）
如果您有Python环境，也可以直接运行源码：
```bash
# 激活虚拟环境
source streamlit_env/bin/activate

# 运行应用
streamlit run streamlit_app.py
```

## 📊 Excel数据格式要求

### 必需列名
您的Excel文件必须包含以下三列（列名必须完全匹配）：
- **九宫格**：档位信息（1-9的数字）
- **员工姓名**：员工的姓名
- **部门**：员工所在部门

### 数据示例
| 九宫格 | 员工姓名 | 部门 |
|--------|----------|------|
| 1      | 张三     | 销售部 |
| 2      | 李四     | 技术部 |
| 5      | 王五     | 销售部 |
| 9      | 赵六     | 人事部 |

### 注意事项
- Excel文件可以包含其他列，系统会自动识别需要的列
- 列的顺序可以任意排列
- 确保关键列中没有空值
- 支持 .xlsx 和 .xls 格式

## 🎯 九宫格布局说明

系统按以下布局显示数据：
```
[7] [8] [9]
[4] [5] [6]
[1] [2] [3]
```

每个格子会显示：
- 标题：档位编号 + 人数统计
- 内容：该档位下所有员工的"姓名(部门)"信息

## 🔧 故障排除

### 常见问题
1. **应用无法启动**
   - 确保整个文件夹完整复制，不要只复制可执行文件
   - 检查系统权限，可能需要允许运行未知来源的应用

2. **浏览器没有自动打开**
   - 手动打开浏览器访问显示的地址（通常是 http://localhost:8501）

3. **Excel文件读取失败**
   - 检查列名是否完全匹配：九宫格、员工姓名、部门
   - 确保文件格式为 .xlsx 或 .xls
   - 检查是否有空值或格式错误

4. **数据显示不正确**
   - 确认"九宫格"列的数据为1-9的数字
   - 检查是否有重复或无效的档位数据
