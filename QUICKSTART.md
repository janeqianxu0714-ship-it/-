# 🚀 快速开始指南

## 一键启动（推荐）

### 方法一：完整安装启动（推荐）
```bash
# macOS/Linux
./setup_and_run.sh

# Windows
setup_and_run.bat
```
此方法会：
- 创建独立的虚拟环境
- 自动安装所有依赖
- 启动应用并打开浏览器

### 方法二：简单启动
```bash
# macOS/Linux
./start.sh

# Windows  
start.bat
```

### 方法三：手动启动
```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 启动应用
streamlit run streamlit_app.py
```

## 📊 使用步骤

1. **启动应用** - 运行上述命令之一
2. **上传Excel** - 在网页中选择您的Excel文件
3. **查看结果** - 系统自动生成九宫格展示

## 📋 Excel格式要求

确保您的Excel文件包含这三列：

| 九宫格 | 员工姓名 | 部门 |
|--------|----------|------|
| 1      | 张三     | 销售部 |
| 5      | 李四     | 技术部 |
| 9      | 王五     | 人事部 |

## 🎯 九宫格布局

```
[7] [8] [9]
[4] [5] [6] 
[1] [2] [3]
```

## ❓ 遇到问题？

1. **依赖安装失败**：确保有Python 3.7+和pip
2. **找不到文件**：检查Excel列名是否正确
3. **应用无法启动**：尝试重新安装依赖
