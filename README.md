# 九宫格潜力展示系统

S1人才盘点九宫格分析工具，支持Excel数据上传、自动部门-负责人映射、九宫格分析和PPT报告生成。

## 功能特点

- 📊 **九宫格潜力分析**：自动生成3x3九宫格人才分布图
- 👥 **梯队统计**：前5%、前15%、前40%、末5%梯队分析
- 🏢 **部门映射**：内置部门-负责人自动匹配功能
- 📈 **数据提取**：从"当前组织"字段智能提取部门信息
- 📋 **PPT报告**：一键生成腾讯风格PPT报告
- 🔧 **灵活配置**：支持自定义Excel列名映射

## 支持的Excel格式

系统支持以下列名（可在代码中配置调整）：
- `日常盘点-九宫格结果（需输入）` - 九宫格评级（1-9）
- `员工姓名` - 员工姓名
- `当前组织` - 组织架构信息（自动提取部门）
- `日常盘点-梯队结果（需输入）` - 梯队评级
- `管理职级` - 管理层级（可选）
- `专业职级` - 专业层级（可选）

## 快速开始

### 本地运行

1. 安装依赖：
```bash
pip install -r requirements.txt
```

2. 运行应用：
```bash
streamlit run streamlit_app.py
```

3. 打开浏览器访问：http://localhost:8501

### 云端部署

推荐使用 [Streamlit Community Cloud](https://share.streamlit.io) 进行免费部署：

1. 将代码上传到GitHub仓库
2. 访问 https://share.streamlit.io
3. 连接GitHub账号，选择仓库
4. 自动部署，获得公开访问链接

## 使用说明

1. **示例数据测试**：勾选"使用示例数据进行测试"查看演示效果
2. **上传真实数据**：上传包含必需列的Excel文件
3. **查看分析结果**：系统自动生成九宫格分布和梯队统计
4. **下载报告**：支持Excel和PPT格式报告下载

## 配置说明

### 列名映射配置
在 `streamlit_app.py` 文件顶部可以调整Excel列名映射：

```python
COLUMN_MAPPING = {
    '日常盘点-九宫格结果（需输入）': '档位',
    '员工姓名': '姓名',
    '当前组织': '当前组织',
    '日常盘点-梯队结果（需输入）': '梯队',
    '管理职级': '管理职级',
    '专业职级': '专业职级'
}
```

### 部门-负责人映射
系统内置了完整的部门-负责人映射表，支持自动匹配。

## 技术栈

- **前端框架**：Streamlit
- **数据处理**：Pandas
- **Excel处理**：OpenPyXL
- **PPT生成**：python-pptx
- **部署平台**：Streamlit Community Cloud
