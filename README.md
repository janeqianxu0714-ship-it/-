# 九宫格潜力展示系统

一个基于Streamlit的员工潜力数据可视化工具，能够将Excel数据以直观的3x3九宫格形式展示。

## 🎯 功能特色

- 📊 **Excel数据导入**: 支持.xlsx和.xls格式
- 🎨 **九宫格展示**: 专业的3x3网格布局
- 👥 **智能分组**: 按部门自动分组员工信息
- 📈 **实时统计**: 动态显示各档位人数
- 🎪 **响应式设计**: 适配不同屏幕尺寸
- 🔒 **数据安全**: 完全本地处理，无数据上传

## 🚀 快速开始

### 环境要求
- Python 3.7+
- pip包管理器

### 安装步骤

1. **克隆项目**
```bash
git clone <your-repo-url>
cd 九宫格潜力展示系统
```

2. **安装依赖**
```bash
pip install -r requirements.txt
```

3. **运行应用**
```bash
streamlit run streamlit_app.py
```

4. **访问应用**
浏览器会自动打开，或手动访问 `http://localhost:8501`

## 📊 数据格式要求

### Excel文件结构
您的Excel文件必须包含以下三列（列名必须完全匹配）：

| 列名 | 数据类型 | 说明 |
|------|----------|------|
| 九宫格 | 数字 | 1-9的档位编号 |
| 员工姓名 | 文本 | 员工姓名 |
| 部门 | 文本 | 所在部门 |

### 示例数据
```csv
九宫格,员工姓名,部门
1,张三,销售部
2,李四,技术部
5,王五,销售部
9,赵六,人事部
```

## 🎨 九宫格布局

系统按以下布局显示数据：
```
[7] [8] [9]
[4] [5] [6]
[1] [2] [3]
```

每个格子显示：
- **标题**: 档位编号 + 人数统计
- **内容**: 员工信息（格式：姓名(部门)）

## 📱 使用说明

## 🛠️ 故障排除

### 常见问题

**Q: 提示"找不到指定列"**
A: 检查Excel文件列名是否为：九宫格、员工姓名、部门

**Q: 应用启动失败**
A: 确保已安装所有依赖：`pip install -r requirements.txt`

**Q: 数据显示不正确**
A: 确认"九宫格"列数据为1-9的数字，无空值

**Q: 浏览器没有自动打开**
A: 手动访问终端显示的地址（通常是 http://localhost:8501）
